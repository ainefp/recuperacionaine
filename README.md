# 📋 RecuperaciónAine

> Aplicación web SPA desarrollada con Vue 3 y Vite para la gestión de empleados, tareas y estadísticas.

---

| [Bootstrap](https://getbootstrap.com/) | ^5.3.8 |

## 📦 Instrucciones de instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/ainefp/recuperacionaine.git
cd recuperacionaine
```

### 2. Instalar dependencias

```bash
npm install
```

### 3. Iniciar el servidor de desarrollo

```bash
npm run dev
```

### 4. Compilar para producción

```bash
npm run build
```

### 5. Previsualizar la build de producción

```bash
npm run preview
```
